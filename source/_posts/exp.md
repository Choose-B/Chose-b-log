# 科学筛选志愿
坏消息：高考志愿第一眼看上去远比中考志愿复杂得多。根据[浙江省2025年普通高校招生普通类第一段平行投档分数线表](https://www.zjzs.net/attach/0/c4110ef9c01a4b6ba1e231c2b5d2462f.xls),2025年的考生共有**17890**个志愿可供填报，能够填入到平行录取志愿的数量是**80**个  
哪个考生能把所有志愿都看完啊  
因此，你最好的办法是从官网上下载这张表，先经过几轮筛选才能进一步找出自己感兴趣的专业  
{% note danger %}
**"可是我是26届的，我不应该看26年的分数线表吗"**  
~~这家伙到底在说什么呢~~   
该分数线表是当年考生志愿填报阶段结束后浙江教育考试院根据录取情况整理的表格    
也就是说，你没填志愿的时候，你这一年的分数线表是无法无中生有的。  
因此每年的考生只能参考前面几年的分数线表来填报志愿  
{% endnote %}  

用excel打开分数线表，全选表格内容，右上角找到**排序和筛选**，打开筛选  
此时所有列标题栏旁边都会出现一个小箭头，点开即可进行筛选的工作  

## 排除 去不了/不想去 的志愿
虽然志愿总数有1w7之多，但是其中的大部分和你没有任何关系——你的高考分数决定了你有肯定考不上的志愿，也有你不屑一顾的志愿  
因此第一步是根据你的分数排除这一部分的志愿  
在未出分的时候，可以自己预估一个分数区间，区间长度尽可能地大，获得一个分数区间长度为60分~100分区间  
以我为例，未出分数时我预估我的分数为`[600,670]`，因此我选出`分数线`栏中在这一区间的志愿  
![打开自定义数字筛选](/img/highschool/1.png)
![设置分数上下限](/img/highschool/2.png)
这样一来，*我可能会考虑的志愿*数量就降到了4k+  
在出分后可以进一步缩小区间，此时上限（下限）建议取自己高考分数+（-）15~20分  
依旧以我为例，我的高考分数恰好为640分，因此我取分数区间`[620,660]`，此时志愿数还有2k+  

根据分数筛选后，就可以根据`学校`和`专业`筛选了  
对于很多同学来说，“自己想要读什么专业”这个问题是令人摸不着头脑的  
如果换一个提问方式，“自己**不想要**读什么学校/专业？”，回答可能就比较清晰了  
打开`专业名称`栏进行筛选，浏览这里出现的所有专业，明确你这次筛选的目的——**排除自己不想去的专业**  
待你花大功夫完成这一步之后，还留在你的考虑列表里面的专业很可能只有几百个了  

{% note danger %}
**如果你是文科生（未选物理+化学）**  
显然，你无法报考带有物化科目要求的专业，你需要从你的excel中排除掉这些你**去不了**的专业
{% endnote %} 

## 查询校园环境
完成第一轮的筛选后，待你考虑的大学大约有几十个，每个大学的校园环境各不相同  
如果不想住到~~难民房~~较差的宿舍/吃到难吃的饭菜/... ，可以访问[大学生生活质量指北](https://colleges.chat/)  
该网站收集了相当数量的本校学生对自己学校生活的各个角度的评价，相对各校官方给出的展示会更加真实  
由此你又能够排除一批学校  

## 根据竞赛队水平筛选
{% note danger %}

{% endnote %} 
如果你对自己未来的规划较为清晰，或许可以考虑根据和你规划方向较为契合的相关竞赛来筛选实力强劲的学校

# 根据意愿排序
经过上面的筛选后，你待考虑的志愿列表里面就只剩下1k不到(如果比1k多可以考虑出分后再进一步筛选或者缩小分数区间)  
想必各位同学会听来各自建议，比如专业优先或者学校优先等  
和各种“志愿填报师”的绝对化的*某某优先最优论*，我的想法是：**如何确定优先级顺序取决于你的个人情况**  

开始排序前，你不妨全选排序后的结果，复制到一张新的sheet内，进行进一步的选择  

## 专业优先
**特点**：优先考虑专业和个人意愿的匹配情况、专业毕业就业情况、专业热门情况，认为专业的排序是第一优先级  
**适合谁**：
+ 拥有较为清晰的专业目标的同学。可以是特别希望去某个特定专业的同学，也可以是对某一个专业群明显感兴趣的同学
+ 自认为在目标的专业/专业群有才能的同学  

**不适合谁**：
+ 对今后就业方向尚不清晰的同学  
+ 对卷绩点、大一转专业、拿保研名额此类事情特别有信心的同学

**常见误区**：
+ 专业优先是优先考虑自己的兴趣方向来选取专业，而不是不自知未来方向盲目追求某一部分“热门专业”  
  **人是为自己而活的**。你选专业是为自己而选的，而不是任何其他人  
+ 当下的“热门专业”是专业选取的最优解  
  土木工程曾经也是热门专业  
  没有人能够预知四年甚至更久之后的社会风潮和就业情况，小心当下的“热门”成为你毕业时的“冷门”  

## 院校优先  
**特点**：优先考虑学校层级、保研率等信息，认为院校的排序是第一优先级。  
**适合谁**:
+ 能够接受的专业面较广，对多类别的专业都接受
+ 或者有充足信心成为绩点王，通过转专业来实现高考分数的以小博大  
+ 追求保研，通常情况下学校层级越高保研率越高

**不适合谁**：
+ 有极其明确目标，不希望大一在不感兴趣的专业上浪费时间的同学

## 地域优先
**特点**：优先考虑学校的地理位置，如是否处于一线城市  
**适合谁**：
+ 看好某城市未来的就业形势，毕业后在同城市工作的同学
+ 或者家庭能够在某城市提供扶持的同学

**不适合谁**：
+ 保研考研意愿强的同学。研究生未必在同校就读。当然如果学校有保研可以考虑保本校，这样该条规则不适应于你了  
+ 不愿意接受一线城市经济压力、生活节奏的同学  

# 检查
不论你以何种方式筛选、排序出一份你心仪的志愿名单，你都应该确保以下几点：
1. 你的志愿排序可以不是严格的分数递减，但是至少要在大体上保持分数的递减趋势，并且存在一个模糊的“冲、稳、保”分层  
   + 冲 ： 录取分数比你的高考分数高出5-10分的志愿，赌运气用的
   + 稳 ： 录取分数在你高考分数±3分的志愿，你大概率被录取的
   + 保 ： 录取分数比你的高考分数低8-20分的志愿，确保你不会滑档的
2. 80个平行志愿可以不填满，但是你最好填入不少于30个  
   不想再体验一次高中备考生活的话就请认真对待  
3. 身边的老师、家长、同学可以给你建议，但是志愿填报的最终决定权在你手中  
   人为自己而活，同学
